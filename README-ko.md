## 선언

많은 프로젝트가 자신들의 도움말/설명 페이지에서 이 가이드를 링크합니다. 그렇게 하는 것은 좋은 일이며, 우리도 모두가 그렇게 하기를 권장합니다. 하지만 만약 이 프로젝트 웹 페이지를 관리하는 사람이라면, 하이퍼링크 근처의 눈에 띄는 자리에 다음을 밝히 표시해 주세요:

**본 가이드는 이 프로젝트의 실제 지원 서비스를 제공하지 않습니다!**
